# Media Samples

CC0 (public domain) audio samples for daStrudel examples.

## Drums — SamsSonor

Sam Greene's Sonor Force 3001 drumkit.
Source: https://github.com/sfzinstruments/SamsSonor
License: CC0 1.0 Universal (Public Domain)

Contains: bd (kick), sd (snare), hh (hihat), cp (crash), ride

## Piano — jRhodes3c

1977 Rhodes Mark I Stage 73 electric piano by jlearman.
Source: https://github.com/sfzinstruments/jlearman.jRhodes3c
License: CC0 1.0 Universal (Public Domain)

Contains: mono FLAC samples, MIDI notes 50-71 (D3-B4), multiple velocity layers.
