Returns the name of the operating system the program is currently running on, such as `"windows"`, `"linux"`, `"darwin"`, `"emscripten"`, or `"unknown"`. Unlike :ref:`get_platform_name <function-builtin_get_platform_name>`, this is evaluated at runtime and never constant-folded at compile time, so in a wasm cross-compiled binary it reports the actual target platform (`"emscripten"`) rather than the compile host. Use it for runtime "where am I executing" decisions, and `get_platform_name` for compile-time or macro decisions.
