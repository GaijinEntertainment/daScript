Queue one frame on ``writer``. ``stride_bytes`` is the pixel-row stride and must be at least ``width * channels`` (negative / bottom-up stride is not supported — flip in the source buffer). ``delay_ms`` is how long the frame is shown. Returns 1 on success, 0 only if the input is invalid (null pixels / bad stride). When the encoder's bounded queue is full this still returns 1 and silently drops the oldest queued frame — call ``stbi_apng_dropped`` to read the running drop count.
