Parses a JSON ``string`` and writes the result into the memory at ``addr`` using the supplied ``TypeInfo``. Equivalent to :ref:`sscan_json <function-builtin-sscan_json>`, but takes an explicit ``(addr, TypeInfo&)`` pair instead of an ``any`` destination, so it can be called from sites that hold only a ``void?`` and a TypeInfo pointer (e.g. typed dispatch tables) without a per-call-site daslang thunk. Returns ``true`` on successful parse.
