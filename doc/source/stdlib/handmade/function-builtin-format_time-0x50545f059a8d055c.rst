Formats a ``clock`` value as local-time text via ``strftime``. The ``format`` string uses ``strftime`` conversion specifiers (e.g. ``%Y-%m-%d %H:%M``); an empty ``format`` falls back to the default ``"%Y-%m-%d %H:%M"``. This is the caller-controlled companion to the ``clock`` type's default ``%c`` rendering. Reads the process locale and timezone; a failed conversion yields an empty string.
